네이버 블로그용 요약본 (copy-paste 자료)
==========================================

목적: 한국인 독자는 네이버에서 검색합니다. 네이버는 외부 사이트보다
네이버 블로그/카페 글을 검색 상단에 노출하므로, 여기 요약본을 네이버
블로그에 올려 검색에 걸리게 하고 → "전문 보기" 링크로 본 사이트
(expatnote.cloud)로 방문을 흘리는 전략입니다.

사용법
------
1. 네이버 블로그를 하나 만듭니다 (기존 블로그 있으면 그거 사용).
2. 아래 각 .txt 파일의 "본문" 부분을 네이버 글쓰기에 붙여넣습니다.
3. 제일 위 "추천 제목"을 글 제목으로 씁니다.
4. 맨 아래 "추천 태그"를 네이버 태그란에 넣습니다.
5. 발행. 2~3일 간격으로 하나씩 올리면 자연스럽습니다.

중요 원칙
---------
- 본문을 사이트 원문과 100% 똑같이 올리지 마세요(중복 콘텐츠 감점).
  여기 요약본은 이미 각도를 틀어 재작성돼 있으니 그대로 쓰면 됩니다.
- 링크(전문 보기)는 반드시 남기세요. 그게 트래픽 유입 통로입니다.
- 사진 1~2장(직접 찍은 것)을 넣으면 네이버 노출에 유리합니다.
- 카페(해외생활/주재원 카페)에 답글로 링크할 때는 홍보 티 나지 않게,
  질문에 실제로 답하면서 "제가 정리한 글이 있어요" 식으로.
